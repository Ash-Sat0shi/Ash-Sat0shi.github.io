• 同時多人数用　検温カメラ

    カメラの前を通行するだけで体温が高い人をスクリーニングすることが可能です。 機器概要
    一式　700,000円　送料　20,000円　※左記はカメラとサイネージ一体型の価格です。現場での導入の際は、希望の月数で分割した価格にてレンタルとすることが可能です。 価格
      同時に30人までリアルタイムで識別し、顔の温度の測定が可能。
      ・顔を認識し計測するため、熱い飲み物等、他の熱源に惑わされず体 温を測ることができる。
      ・しきい値、警告を鳴らす温度を設定し、それ以上の体温の者がカメラ に写った場合に音と光により、その場で警告。同時にメール等でのお知ら せが可能。（警告時の画像が添付される）
      ・インターネットに接続することで遠隔地からライブ映像を見ることが可能。
      ・カメラ本体とサイネージを分離する等、設置方法変更の対応が可能。
    		機能
    ・受付等、監視する人がいる場所への設置推奨。異常体温者が通った際、警告対象が自分と気づかなかった場合は、監視人等が知らせる必要がある。
    ・顔を識別するが、個人を特定（顔認証）する機能はない。    		 注意事項
    可視光映像 : 2688 x 1520 px
    サーマル映像 : 1280 x 720 px
    測定誤差 : ±0.5℃ / ±0.3℃（別売りブラックボディにて校正した場合）
    温度測定範囲 : 30℃～45℃
    同時体温測定上限 : 枠内30人まで    		 スペック





  • 顔認証AI/サーマルカメラ

    機器概要 AIで個人を特定し、図った体温をデータベースに記録することが可能です。
    価格 レンタル 1年目 24,000円/月（サポート付き）
    2年目 9,800円/月（サポート付き）
    機能 ・しきい値、警告を鳴らす温度を設定し、それ以上の体温の者がカメラ に写った場合に音と画面メッセージにより、その場で警告。
    ・顔認証機能により、人物の特定、ログの記録確認が可能。
    ・顔認証機能では写真や動画での「なりすまし」偽生体を防止。
    ・マスクを着用しているかどうかを自動AI認識し、マスクを着用していな い人にアラートを発することが可能。
    ※マスク着用は口、鼻の一部がカバーされているかを識別します。
    マスクを顎に着けていて口が出ている場合はNG判定となります。
    ・本カメラの顔認証機能とBuildeeとの連携について開発検討を進め ており、将来的にはCCUS入場管理も可能となる予定。
    注意事項 ・顔の登録を事前に行わないと、個人の特定はできない。
    ※検温、マスク着用有無は、顔の登録をしなくても利用可能。
    ログ記録には名前なしで顔写真のみ記録される。
    スペック ディスプレイ解像度: 1024 x 600 px
    測定誤差 : ±0.3℃
    体温測定範囲 : 32℃～41℃
    体温測定距離：0.3ｍ～1.5m
    保存容量：顔データベース3万人、15万件レコード
